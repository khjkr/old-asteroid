## Installation
Dependencies
  - discord.js
  - express
  - mysql
  - typescript
  - colors
  - ts-node
  - etc...

1. Rename `const.inc.json` to `const.json`
2. Edit `const.json`
3. Run `db.sql` to your MariaDB Server
4. `yarn start` or `npm start`
